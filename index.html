welcome to Ravi Kishore Thella's Weather App

This is a web-based application which allows the users to search for the current weather inforamtion of different cities across the world. This app uses the openweathermap API which can be found at www.openweathermap.org

Weather API

Use openweathermap API to get the weather inforamtion of any city.

Search

Search for current weather inforamtion and weather forecast of any city.

View Result API

Get inforamtion about Temperature, Humidity, Pressure etc, of any city.
